# Stock Profit Calculator

DSAI 2018 HW1 corrector

## Prerequisite
- Python 3.6
- [pipenv](http://pipenv.org)

### Install Dependency
```sh
pipenv install
```

### Start Virtual Environment
```sh
pipenv shell
```


## Usage
```sh
python corrector.py [-h] stock action
```

- stock: Input stock file
- action: Input action file
